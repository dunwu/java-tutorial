# vscode 应用指南

## 快捷键

### 命令面板（Command Palette）

根据您当前的上下文访问所有可用命令。

> Mac: cmd+shift+p or f1
> Windows / Linux: ctrl+shift+p or f1

### 快速打开文件（Quick Open）

> Mac: cmd+p
> Windows / Linux: ctrl+p

### Status Bar

> Mac: shift+cmd+m
> Windows / Linux: ctrl+shift+m

### 改变语言模式

> Mac: cmd+k m
> Windows / Linux: ctrl+k m

### 设置

> Mac: cmd+,
> Windows / Linux: **File** > **Preferences** > **Settings** or ctrl+,
