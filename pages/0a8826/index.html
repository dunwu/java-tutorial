Zipkin еҝ«йҖҹе…Ҙй—Ё

# Zipkin еҝ«йҖҹе…Ҙй—Ё

Zipkin жҳҜдёҖдёӘеҹәдәҺ Java ејҖеҸ‘зҡ„гҖҒејҖжәҗзҡ„гҖҒеҲҶеёғејҸе®һж—¶ж•°жҚ®и·ҹиёӘзі»з»ҹпјҲDistributed Tracking SystemпјүгҖӮе®ғйҮҮйӣҶжңүеҠ©дәҺи§ЈеҶіжңҚеҠЎжһ¶жһ„дёӯе»¶иҝҹй—®йўҳзҡ„е®һж—¶ж•°жҚ®гҖӮ

Zipkin дё»иҰҒеҠҹиғҪжҳҜиҒҡйӣҶжқҘиҮӘеҗ„дёӘејӮжһ„зі»з»ҹзҡ„е®һж—¶зӣ‘жҺ§ж•°жҚ®гҖӮеҲҶеёғејҸи·ҹиёӘзі»з»ҹиҝҳжңүе…¶д»–жҜ”иҫғжҲҗзҶҹзҡ„е®һзҺ°пјҢдҫӢеҰӮпјҡNaver зҡ„ PinpointгҖҒApache зҡ„ HTraceгҖҒйҳҝйҮҢзҡ„й№°зңј TracingгҖҒдә¬дёңзҡ„ HydraгҖҒж–°жөӘзҡ„ WatchmanпјҢзҫҺеӣўзӮ№иҜ„зҡ„ CATпјҢskywalking зӯүгҖӮ

Zipkin еҹәдәҺ Google Dapper зҡ„и®әж–Үи®ҫи®ЎиҖҢжқҘпјҢз”ұ Twitter е…¬еҸёејҖеҸ‘иҙЎзҢ®гҖӮ

# дёҖгҖҒZipkin з®Җд»Ӣ

# зү№жҖ§

еҰӮжһңж—Ҙеҝ—ж–Үд»¶дёӯжңүи·ҹиёӘ IDпјҢеҲҷеҸҜд»ҘзӣҙжҺҘи·іиҮіиҜҘи·ҹиёӘ IDгҖӮ еҗҰеҲҷпјҢжӮЁеҸҜд»ҘеҹәдәҺеұһжҖ§иҝӣиЎҢжҹҘиҜўпјҢдҫӢеҰӮжңҚеҠЎпјҢж“ҚдҪңеҗҚз§°пјҢж Үзӯҫе’ҢжҢҒз»ӯж—¶й—ҙгҖӮ е°ҶдёәжӮЁжҖ»з»“дёҖдәӣжңүи¶Јзҡ„ж•°жҚ®пјҢдҫӢеҰӮеңЁжңҚеҠЎдёӯиҠұиҙ№зҡ„ж—¶й—ҙзҷҫеҲҶжҜ”д»ҘеҸҠж“ҚдҪңжҳҜеҗҰеӨұиҙҘгҖӮ

Zipkin UI иҝҳжҸҗдҫӣдәҶдёҖдёӘдҫқиө–е…ізі»еӣҫпјҢиҜҘе…ізі»еӣҫжҳҫзӨәдәҶжҜҸдёӘеә”з”ЁзЁӢеәҸдёӯи·ҹиёӘдәҶеӨҡе°‘дёӘиҜ·жұӮгҖӮиҝҷеҜ№дәҺиҜҶеҲ«иҒҡеҗҲиЎҢдёәпјҲеҢ…жӢ¬й”ҷиҜҜи·Ҝеҫ„жҲ–еҜ№дёҚиөһжҲҗдҪҝз”Ёзҡ„жңҚеҠЎзҡ„и°ғз”ЁпјүеҫҲжңүеё®еҠ©гҖӮ

# еӨҡе№іеҸ°

Zipkin е®ҳж–№ж”ҜжҢҒ C#гҖҒGoгҖҒJavaгҖҒJavaScriptгҖҒRubyгҖҒScalaгҖҒPHP иҜӯиЁҖгҖӮ

йҷӨжӯӨд»ҘеӨ–пјҢзӨҫеҢәиҝҳиҙЎзҢ®дәҶеӨҡз§Қе…¶д»–иҜӯиЁҖзҡ„ж”ҜжҢҒпјҢиҜҰжғ…еҸҜд»ҘеҸӮиҖғе®ҳж–№ж–ҮжЎЈпјҡTracers and Instrumentation (opens new window)

# ж•°жҚ®

Zipkin жңҚеҠЎеҷЁжҚҶз»‘дәҶз”ЁдәҺйҮҮйӣҶе’ҢеӯҳеӮЁж•°жҚ®зҡ„жү©еұ•гҖӮ

й»ҳи®Өжғ…еҶөдёӢпјҢж•°жҚ®еҸҜд»ҘйҖҡиҝҮ HttpпјҢKafka гҖҒRabbitMQ жҲ– RPC дј иҫ“гҖӮ

е№¶еӯҳеӮЁеңЁеҶ…еӯҳдёӯжҲ– MySQLгҖҒCassandra жҲ– Elasticsearch дёӯгҖӮ

ж•°жҚ®д»Ҙ json еҪўејҸеӯҳеӮЁпјҢеҸҜд»ҘеҸӮиҖғпјҡZipkin е®ҳж–№зҡ„ Swagger API (opens new window)

# дәҢгҖҒZipkin е®үиЈ…

# Docker

Docker еҗҜеҠЁж–№ејҸпјҡ

docker run -d -p 9411:9411 openzipkin/zipkin

# Java

жіЁж„Ҹпјҡеҝ…йЎ»иҝҗиЎҢеңЁ JDK8+ зҺҜеўғ

Java еҗҜеҠЁж–№ејҸпјҡ

curl -sSL https://zipkin.io/quickstart.sh | bash -s
java -jar zipkin.jar

# зј–иҜ‘ж–№ејҸ

йҖӮз”ЁдәҺйңҖиҰҒи®ўеҲ¶еҢ–зҡ„еңәжҷҜгҖӮ

# get the latest source
git clone https://github.com/openzipkin/zipkin
cd zipkin
# Build the server and also make its dependencies
./mvnw -DskipTests --also-make -pl zipkin-server clean install
# Run the server
java -jar ./zipkin-server/target/zipkin-server-*exec.jar

# дёүгҖҒZipkin жһ¶жһ„

ZipKin еҸҜд»ҘеҲҶдёәдёӨйғЁеҲҶпјҢ

  • дёҖйғЁеҲҶжҳҜ Zipkin serverпјҢз”ЁжқҘдҪңдёәж•°жҚ®зҡ„йҮҮйӣҶеӯҳеӮЁгҖҒж•°жҚ®еҲҶжһҗдёҺеұ•зӨәпјӣ
  • еҸҰдёҖйғЁеҲҶжҳҜ Zipkin client жҳҜ Zipkin еҹәдәҺдёҚеҗҢзҡ„иҜӯиЁҖеҸҠжЎҶжһ¶е°ҒиЈ…зҡ„дёҖдәӣеҲ—е®ўжҲ·з«Ҝе·Ҙе…·пјҢиҝҷдәӣе·Ҙе…·е®ҢжҲҗдәҶиҝҪиёӘж•°жҚ®зҡ„з”ҹжҲҗдёҺдёҠжҠҘеҠҹиғҪгҖӮ

жһ¶жһ„еҰӮдёӢпјҡ

# Zipkin Server

Zipkin Server дё»иҰҒеҢ…жӢ¬еӣӣдёӘжЁЎеқ—пјҡ

  • Collector - иҙҹиҙЈйҮҮйӣҶе®ўжҲ·з«Ҝдј иҫ“зҡ„ж•°жҚ®гҖӮ
  • Storage - иҙҹиҙЈеӯҳеӮЁйҮҮйӣҶзҡ„ж•°жҚ®гҖӮеҪ“еүҚж”ҜжҢҒ MemoryпјҢMySQLпјҢCassandraпјҢElasticSearch зӯүпјҢй»ҳи®ӨеӯҳеӮЁеңЁеҶ…еӯҳдёӯгҖӮ
  • APIпјҲQueryпјү - иҙҹиҙЈжҹҘиҜў Storage дёӯеӯҳеӮЁзҡ„ж•°жҚ®гҖӮжҸҗдҫӣз®ҖеҚ•зҡ„ JSON API иҺ·еҸ–ж•°жҚ®пјҢдё»иҰҒжҸҗдҫӣз»ҷ web UI дҪҝз”ЁгҖӮ
  • UI - жҸҗдҫӣз®ҖеҚ•зҡ„ web з•ҢйқўгҖӮ

Instrumented Client е’Ң Instrumented ServerпјҢжҳҜжҢҮеҲҶеёғејҸжһ¶жһ„дёӯдҪҝз”ЁдәҶ Trace е·Ҙе…·зҡ„дёӨдёӘеә”з”ЁпјҢClient дјҡи°ғз”Ё Server жҸҗдҫӣзҡ„жңҚеҠЎпјҢдёӨиҖ…йғҪдјҡеҗ‘ Zipkin дёҠжҠҘ Trace зӣёе…ідҝЎжҒҜгҖӮеңЁ Client е’Ң Server йҖҡиҝҮ Transport дёҠжҠҘ Trace дҝЎжҒҜеҗҺпјҢз”ұ Zipkin зҡ„ Collector жЁЎеқ—жҺҘж”¶пјҢе№¶з”ұ Storage жЁЎеқ—е°Ҷж•°жҚ®еӯҳеӮЁеңЁеҜ№еә”зҡ„еӯҳеӮЁд»ӢиҙЁдёӯпјҢз„¶еҗҺ Zipkin жҸҗдҫӣ API дҫӣ UI з•ҢйқўжҹҘиҜў Trace и·ҹиёӘдҝЎжҒҜгҖӮNon-Instrumented ServerпјҢжҢҮзҡ„жҳҜжңӘдҪҝз”Ё Trace е·Ҙе…·зҡ„ ServerпјҢжҳҫз„¶е®ғдёҚдјҡдёҠжҠҘ Trace дҝЎжҒҜгҖӮ

# Zipkin Client

  • Tracer - Tracer еӯҳеңЁдәҺдҪ зҡ„еә”з”ЁдёӯпјҢе®ғиҙҹиҙЈйҮҮйӣҶе…ідәҺе·ІеҸ‘з”ҹж“ҚдҪңзҡ„е®һж—¶е…ғж•°жҚ®гҖӮе®ғд»¬йҖҡеёёдјҡжЈҖжөӢеә“пјҢеӣ жӯӨеҜ№дәҺз”ЁжҲ·жҳҜйҖҸжҳҺзҡ„гҖӮдҫӢеҰӮпјҢе·ІжЈҖжөӢзҡ„ Web жңҚеҠЎеҷЁи®°еҪ•е®ғдҪ•ж—¶жҺҘж”¶еҲ°иҜ·жұӮпјҢд»ҘеҸҠдҪ•ж—¶еҸ‘йҖҒе“Қеә”гҖӮж”¶йӣҶзҡ„и·ҹиёӘж•°жҚ®з§°дёәи·ЁеәҰпјҲSpanпјүгҖӮ
  • Instrumentation - Instrumentation дҝқиҜҒдәҶз”ҹдә§зҺҜеўғзҡ„е®үе…ЁжҖ§е’ҢеҫҲе°‘зҡ„ејҖй”ҖгҖӮеӣ жӯӨпјҢе®ғд»¬д»…еңЁеҶ…йғЁдј ж’ӯ IDпјҢд»Ҙе‘ҠзҹҘжҺҘж”¶ж–№жӯЈеңЁиҝӣиЎҢиҝҪиёӘгҖӮе®ҢжҲҗзҡ„ Span е°ҶйҖҡиҝҮеӨ–йғЁйҖҡдҝЎе‘ҠзҹҘ ZipkinпјҢзұ»дјјдәҺеә”з”ЁзЁӢеәҸејӮжӯҘжҠҘе‘ҠжҢҮж Үзҡ„ж–№ејҸгҖӮдҫӢеҰӮпјҢеҪ“и·ҹиёӘжҹҗдёӘж“ҚдҪңе№¶дё”йңҖиҰҒеҸ‘еҮә http иҜ·жұӮж—¶пјҢдјҡж·»еҠ дёҖдәӣ header жқҘдј ж’ӯ IDгҖӮheader дёҚз”ЁдәҺеҸ‘йҖҒиҜҰз»ҶдҝЎжҒҜпјҢдҫӢеҰӮж“ҚдҪңеҗҚз§°гҖӮ
  • Reporter - иғҪеӨҹе°Ҷж•°жҚ®еҸ‘йҖҒеҲ° Zipkin зҡ„жЈҖжөӢеә”з”ЁзЁӢеәҸдёӯзҡ„з»„д»¶пјҢиў«з§°дёә ReporterгҖӮReporter жңүеӨҡз§Қдј иҫ“ж–№ејҸпјҢеҸҜд»Ҙе°Ҷи·ҹиёӘж•°жҚ®еҸ‘йҖҒеҲ° Zipkin йҮҮйӣҶеҷЁпјҢеҗҺиҖ…е°Ҷи·ҹиёӘж•°жҚ®жҢҒд№…еҢ–дҝқеӯҳеҲ°еӯҳеӮЁдёӯгҖӮзЁҚеҗҺпјҢAPI дјҡжҹҘиҜўеӯҳеӮЁд»Ҙеҗ‘ UI жҸҗдҫӣжёІжҹ“ж•°жҚ®гҖӮ
